说明


软件不定期同步大神库更新，适合一键下载到package目录下，用于openwrt编译



食用方法一、


在feeds.conf.default加入：

src-git mypackages https://github.com/yjiawqgj/mypackages

食用方法二、

git clone https://github.com/yjiawqgj/mypackages.git package/mypackages
